URL Quote под рукой

08.08.2013 | Comments

Плагин для Sublime Text 3 для быстрого URL quote (превращение “строка” в “%D1%81%D1%82%D1%80%D0%BE%D0%BA%D0%B0”). Здесь используется urllib.parse.quote из python3. Для python2 надо использовать что-то другое.

Tools -> New Plugin …

Сохраняем как url_quote_string.py и назначаем кнопки ctrl+shift+q: Preferences -> Key Bindings - User

Должно работать.
